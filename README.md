# Wazeer Store (وزير العباية ستور)

A premium e-commerce platform built with Next.js 15, Tailwind 4, and Node.js.

## 📁 Project Structure

- **`/frontend`**: Next.js 15 application with App Router, Tailwind 4, and Zustand.
- **`/backend`**: Express.js server with MongoDB and Redis.

## 🤖 AI Agent Guide

If you are an AI assistant working on this project, please read the [AI Agent README](README.agent.md) first. It contains critical information about the brand identity, tech stack, and development guidelines.

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- MongoDB
- Redis

### Installation

1. **Frontend**:

   ```bash
   cd frontend
   npm install
   npm run dev
   ```

2. **Backend**:
   ```bash
   cd backend
   npm install
   npm run dev
   ```
